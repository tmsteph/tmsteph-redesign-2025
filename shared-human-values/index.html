A Shared Human Values

A post-religious, values-first framework for unity, ethics, and human flourishing — without dogma, coercion, or ideology.

Why This Exists

America doesn’t need another religion. It needs a shared moral operating system — one that works whether you believe in God, many gods, or none at all.

This project focuses on what actually changes societies: inner responsibility, ethical behavior, and cooperative action.

The Core Principles

Inner Responsibility

Each person takes responsibility for their inner state, reactions, and growth.

Compassion

We reduce harm and increase care for others, especially the vulnerable.

Truthfulness

We value honesty, reality-based thinking, and intellectual humility.

Non-Coercion

No forced belief, no shaming, no domination — consent and dignity matter.

Service

We translate values into action through service, repair, and contribution.

Shared Humanity

Differences are real, but dehumanization is never acceptable.

What This Is Not

  • Not a religion
  • Not a political party
  • Not a belief test
  • Not anti-faith or anti-science

How It Shows Up

Community Spaces

Quiet rooms, repair cafes, learning hubs, mutual aid groups.

Education

Emotional literacy, ethics, systems thinking, and practical skills.

Technology

Open-source, humane tools that empower rather than extract.

Start Where You Are

You don’t need permission. You don’t need belief alignment. You just need a willingness to live these values.
